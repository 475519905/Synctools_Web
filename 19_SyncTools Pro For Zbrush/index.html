ZBrush 支持

推荐配置:

  • 推荐版本: ZBrush 2023 （或更高版本）

  • 最低兼容版本: ZBrush 2022

  • 支持的操作系统: Windows (64-bit), macOS (64-bit)

功能

SyncTools for ZBrush 支持以下 3D 资源类型的双向导入/导出:

  • 3D 模型几何

  • Subtool 结构

  • FBX 格式数据传输

该插件使用复制和粘贴操作同步资源，大大简化了跨多个 ZBrush 窗口或项目的工作流。

安装指南

  2. 重启 ZBrush, 插件将自动加载。

使用方法


  • 打开插件面板: Alt + Shift + P

快捷键:

  • 复制: 复制当前选中的 Subtool 几何体
    快捷键: Alt + Shift + C

  • 粘贴: 将几何体粘贴到当前 ZBrush 场景中
    快捷键: Alt + Shift + V

注意事项

  • 插件会在后台自动创建临时文件以进行数据交换。

  • 确保有足够的磁盘空间（建议至少 500MB）。

  • 使用插件前请务必保存您的工作，以避免意外数据丢失。

  • 粘贴前必须先执行复制操作。

常见问题及解决方案

插件面板未显示


  • 确保插件文件正确放置在 ZPlugs64/SyncTools 目录中。

  • 检查您的 ZBrush 版本（最低要求：2023）。

  • 重启 ZBrush。

复制失败

  • 确保已选择要复制的 Subtool。

  • 检查是否有足够的磁盘空间。

  • Ensure Subtool名称 not contain special characters (* ? / \ | " < > :).

粘贴失败

  • 确保已先执行复制操作。

  • 检查界面上是否显示"缓冲区为空！"消息。

  • 验证指定系统路径中是否存在临时文件。

数据兼容性

支持的数据类型:

  • ✔ 完全支持:

    • 基本几何网格

    • Subtool 结构

    • FBX 格式数据

    • 顶点 positions and topology

  • ⚠ 部分支持:

    • UV 映射数据

    • 多边形组信息 (Poly分组s)

    • 顶点颜色数据 (Polypaint)

  • ✖ 不支持:

    • ZBrush 材质

    • 多边形组着色

    • 笔刷笔触

    • 图层信息

    • 动态细分设置

重要说明

  • 数据格式: 使用标准 FBX 格式进行数据传输，确保广泛的兼容性。

  • 临时文件管理: 插件自动创建和删除临时交换文件——无需手动干预。

  • 命名规范: 避免在文件名中使用特殊字符，以防止复制/粘贴失败。

  • 系统要求: 该插件仅支持 64 位操作系统环境。

总结

SyncTools for ZBrush 提供了一种简单高效的方式，可在不同 ZBrush 实例或项目之间快速传输 3D 模型数据。 通过一键复制粘贴流程，显著提高了多窗口和跨项目工作流的便利性和生产力。
