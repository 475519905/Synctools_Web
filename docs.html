入门指南

概述

SyncTools 采用统一数据交换协议，支持多 DCC 间的双向互导与跨平台协作。

安装

  • Cinema 4D：R21+；插件目录示例 C:\\Program Files\\Maxon Cinema 4D 2024\\plugins（无目录则新建，建议系统盘）
  • Blender：3.3+；偏好设置 → 从磁盘安装，直接选择压缩包（无需解压）

激活

  • 人工激活：提供订单号，人工下发许可证
  • 邮箱激活（推荐）：发送激活码到 qq475519905@gmail.com（标题为订单号，仅数字），收到许可证后重启软件

首次上手

  • C4D：首次前在 FBX 导出设置勾选“仅选集”
  • 在 C4D 与 Blender 中使用 Copy/Paste 在两端互导对象

功能文档

支持范围

角色、动画、样条、材质、模型、摄像机、灯光、工程设置等资产的双向互导。

核心能力

  • 快速复制与粘贴：自动处理比例、轴向与层级
  • 项目参数同步：焦距、分辨率、宽高比、帧率等
  • 材质互导：支持 Octane、Redshift、V-Ray、Cycles、Arnold 等
  • 灯光互导：强度、颜色、贴图与节点/标签
  • 动画互导：关键帧与曲线手柄保持精度
  • 离线继续导入：跨软件延续导入/导出流程

工程设置同步

Blender：Edit → Preferences → Add-ons → SyncTools → Project Info Sync Settings 选择要导入/导出的分辨率、帧率、帧范围、输出路径、文件格式等字段。

轴向与比例预设

按目标 DCC 自动适配或自定义坐标映射；提供 All / Fast / Medium 等预设以适配不同场景。

使用技巧

  • Blender 行为设置：替换选中、多次导入可提升迭代效率
  • 模式选择：标准（均衡）/ Beta（精度优先）/ 快速（性能优先）
  • 新版模式（实验性）：为 UE、MAX、MAYA、ZB、BL、4D 建立共同剪贴板
  • 识别渲染器：自动创建对应着色器缓存；或切换 RS/标准/物理
  • 快捷键可自定义：粘贴(Shift+Ctrl+C)、复制(Shift+Ctrl+V)、导入材质(Shift+Ctrl+M) 等

常见问题（FAQ）

  • Q：材质不匹配怎么办？ A：确认渲染器一致并开启材质识别；必要时使用标准节点过渡。
  • Q：动画不对齐？ A：检查两端帧率与帧范围设置是否一致。
  • Q：粘贴后比例异常？ A：调整轴向与比例预设，或在导出设置中统一全局缩放。

故障排查

  • 权限问题：尽量安装在系统盘，避免因权限导致自动更新失败。
  • 导出异常：C4D 勾选“仅选集”；检查是否存在只读路径或非法字符。
  • 卡顿/崩溃：逐步排查第三方脚本与大型贴图；缩小导入范围测试。
