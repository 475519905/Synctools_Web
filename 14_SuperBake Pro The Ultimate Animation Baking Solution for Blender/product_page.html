🚀 概述

超级烘焙专业版 是Blender的终极动画烘焙解决方案。它允许您轻松烘焙复杂的动画，包括父子关系、约束和驱动程序。

Blender强大的动画系统可以创建复杂的角色、机械和场景动画。然而，导出这些动画通常会导致父子关系中断、约束缺失或特殊对象失去运动。

Super Bake Pro 解决了这个问题 一键解决方案 that preserves all animation data through an intelligent GLTF-based bake/export workflow.

Super Bake Pro - 用户指南

概述

Super Bake Pro 是 Blender 的终极动画烘焙解决方案。它简化了将复杂动画烘焙为干净、可导出关键帧的过程 - 所有这些都通过一键式工作流程完成。无论您是使用约束、父子关系、驱动程序、形状键还是相机和灯光，Super Bake Pro 都能确保您的动画保持完整。

安装

  1. 下载 .zip 来自 Blender Market 或您首选来源的文件。

  2. 打开 Blender 并转到 Edit > Preferences > Add-ons.

  3. Click Install，选择 .zip 文件，然后单击 Install Add-on.

  4. 启用 超级烘焙专业版 在列表中。

安装后，您可以从 N-panel 根据 超级烘焙专业版 tab.

基本用法

  1. 选择要烘焙的对象。

  2. 设置您的帧范围。

  3. 选择烘焙模式：

    • 选定的对象

    • 可见物体

    • 整个场景

  4. 调整可选设置（例如保留材料、自定义命名）。

  5. Click 烘焙动画.

Super Bake Pro 将处理您的场景、烘焙动画并生成新的烘焙版本 _Baked suffix.

核心烘烤

  • 保持完整的养育和等级制度

  • 烘焙约束和驱动因素

  • 支持相机、灯光和形状关键动画

  • 将所有内容转换为干净的关键帧

烘烤模式
