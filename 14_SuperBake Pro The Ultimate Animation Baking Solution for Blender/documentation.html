Super Bake Pro - 用户指南

概述

Super Bake Pro是Blender的终极动画烘焙解决方案。它简化了将复杂动画烘焙为干净、可导出关键帧的过程——全部通过一键式工作流程完成。无论您使用约束、父子关系、驱动器、形态键，还是摄像机和灯光，Super Bake Pro都能确保您的动画完好无损。

安装

  1. 从Blender Market或您选择的来源下载 .zip 文件。

  2. 打开Blender，前往 编辑 > 偏好设置 > 插件。

  3. 点击 安装，选择 .zip 文件，然后点击 安装插件。

  4. 在列表中启用 Super Bake Pro。

安装完成后，您可以在 N面板 的 Super Bake Pro 标签页中访问它。

基本用法

  1. 选择要烘焙的物体。

  2. 设置帧范围。

  3. 选择烘焙模式：

    • 选中物体

    • 可见物体

    • 完整场景

  4. 调整可选设置（如保留材质、自定义命名）。

  5. 点击 烘焙动画。

Super Bake Pro将处理您的场景、烘焙动画，并生成带有 _Baked 后缀的新烘焙版本。

核心功能

核心烘焙

  • 维护完整的父子关系和层级结构

  • 烘焙约束和驱动器

  • 支持摄像机、灯光和形态键动画

  • 将所有内容转换为干净的关键帧

烘焙模式

  • 选中物体：推荐用于手动控制

  • 可见物体：适用于渲染视图

  • 完整场景：烘焙所有物体

  • 自动模式：未选择物体时使用上下文感知逻辑

高级选项

  • 强制保留子物体动画

  • 摄像机/灯光动画保留

  • 形态键烘焙支持

  • 自定义帧范围设置

数据处理

  • 可选的材质保留

  • 顶点颜色和切线导出

  • 自动添加 _Baked 后缀重命名

文件与命名

  • 随机命名确保唯一性

  • 自动清理临时文件

  • 跨平台路径兼容

支持的物体类型

  • 网格

  • 曲线

  • 曲面

  • 融球

  • 文本物体

  • 骨架（骨骼）

  • 空物体

  • 摄像机

  • 灯光

兼容性

  • Blender版本：3.0+官方正式版

  • 动画类型：关键帧、约束、驱动器、形态键、NLA

  • 插件兼容性：兼容Rigify、Auto-Rig Pro及其他插件

故障排除

  • 错误信息将附带建议的解决方案

  • 操作失败时自动回滚将恢复原始状态

  • 烘焙前请务必检查帧范围和物体选择

获取最佳效果的技巧

  • 尽可能使用干净的父子关系设置

  • 烘焙前应用变换

  • 避免不必要的驱动器重叠

  • 使用针对烘焙数据优化的GLTF导出设置
