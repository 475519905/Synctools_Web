SyncTools for Marvelous Designer

SyncTools for Marvelous Designer

概述

SyncTools Pro 是一款跨软件工具，旨在简化 Marvelous Designer 与其他应用程序之间复制和粘贴详细场景数据的流程。只需一键，即可将对象（包括网格、材质、动画等）从一个场景导出并导入到另一个场景，为您的制作工作流程节省大量时间和精力。

多 DCC 平台支持

SyncTools 系列现已支持主流数字内容创作软件，包括：

  • SyncTools for Maya（已发布）
  • SyncTools for 3ds Max（已发布）
  • SyncTools for Houdini（已发布）
  • SyncTools for Cinema 4D（已发布）
  • SyncTools for Rhino（已发布）
  • SyncTools for Marvelous Designer（已发布）
  • SyncTools for Substance Painter（开发中）
  • SyncTools for Unreal Engine（开发中）
  • SyncTools for Unity（开发中）
  • SyncTools for Zbrush（开发中）

所有插件均采用统一的数据交换协议，实现跨平台无损资产转换和真正的多软件协同工作流程。



安装指南

1. 系统要求

  • 操作系统：Windows 10/11
  • Marvelous Designer 版本：10 或更高版本
  • Blender 版本：3.4 或更高版本

2. 插件安装步骤

下载与解压：

  • 下载 SyncTools_MD.zip 文件
  • 将内容解压到临时目录

在 Marvelous Designer 中安装：

  1. 启动 Marvelous Designer
  2. 点击顶部菜单栏中的 "Script"
  3. 选择 "Python" >"打开文件"
  4. 导航并选择解压后的 SyncTools_MD.py 文件
  5. "Sync Tools" 菜单将出现在顶部菜单栏中

验证安装：

  • 点击 "Sync Tools" 菜单
  • 您应该能看到 "复制" 和 "粘贴" 选项
  • 如果这些选项可见，则安装成功

替代安装（可选）：

  • 您也可以手动将 SyncTools_MD.py 复制到 Marvelous Designer 的脚本目录：
    C:\Users\[您的用户名]\Documents\Marvelous Designer [版本]\Scripts
  • 手动安装后需重启 Marvelous Designer

使用指南

1. 基本工作流程

  • 复制操作：

    • 在场景中选择要复制的对象
    • 从 Sync Tools 菜单点击 "复制"（或使用 Ctrl+Shift+C）
    • 插件将您的选择导出为 FBX 文件到指定的缓存文件夹

  • 粘贴操作：

    • 切换到目标场景
    • 从 Sync Tools 菜单点击 "粘贴"（或使用 Ctrl+Shift+V）
    • 插件自动导入最新的缓存文件

2. 高级功能

  • 支持多对象选择
  • 自动材质转换
  • 变换层级保留
  • 动画数据传输

数据兼容性

完全支持：

  • ✔ 基础几何体（网格/服装）
  • ✔ 变换层级
  • ✔ 核心材质属性
  • ✔ 服装物理属性
  • ✔ 基础动画关键帧
  • ✔ 缝纫线信息

部分支持：

  • ⚠ 复杂着色器节点（自动转换为近似值）
  • ⚠ 服装权重（需匹配顶点顺序）
  • ⚠ UV 贴图（最多支持 4 个通道）
  • ⚠ 服装褶皱数据

不支持：

  • ✖ 粒子系统
  • ✖ 流体模拟数据
  • ✖ 程序化纹理（需预先烘焙）
  • ✖ 自定义着色器网络
  • ✖ 动态约束（刚体/布料）

重要注意事项

1️⃣ 单位系统：建议使用公制（米）
2️⃣ 轴向转换：自动处理 Y-上 ↔ Z-上 转换
3️⃣ 材质映射：自动标准材质转换
4️⃣ 命名规范：特殊字符（@#%）可能导致导入问题
5️⃣ 插件依赖：需要匹配的 SyncTools 版本

  • （数据通过 FBX 中间件转换——请查看发行说明以获取最新兼容性更新）

故障排除

  • 如果菜单未显示，请检查脚本是否在正确的目录中
  • 确保您拥有正确版本的 Marvelous Designer
  • 检查控制台是否有错误信息
  • 确保您在 Documents 文件夹中具有写入权限
