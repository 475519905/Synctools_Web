导入 Adobe Illustrator (.ai) 文件
将 Adobe Illustrator .ai 文件直接导入 Blender，完全支持图像、渐变、颜色和图层组织。这款强大的插件使用内置的 Inkscape 来将 .ai 文件转换为 SVG 格式，确保与 Blender 原生 SVG 导入器的无缝集成。
功能介绍
功能介绍
此插件弥合了 Adobe Illustrator 和 Blender 之间的差距，让您能够将矢量图稿无缝导入 Blender 的 3D 环境中。 无论您在 Illustrator 中创建徽标、插图、技术图纸还是图形设计，现在都可以将它们直接导入 Blender 进行 3D 可视化、动画、纹理制作或进一步的创意工作。

核心功能
核心功能
✓ 零外部依赖
无需安装任何额外软件即可导入 .ai 文件。插件包含完整的内置 Inkscape，所有需要的功能开箱即用。
✓ 全面的文件格式支持
处理基于 PDF 的现代 .ai 文件（Adobe Illustrator CS 及更高版本）和基于 PostScript 的旧版 .ai 文件（旧版本），确保与各版本 Adobe Illustrator 文件的兼容性。
✓ 完整的视觉保真度
保留嵌入的图像、渐变、颜色和矢量路径，与在 Illustrator 中的显示完全一致。您的作品在整个导入过程中保持视觉完整性。
✓ 智能图层管理
自动图层偏移功能通过根据图层顺序添加小的 Z 轴偏移来防止几何体重叠。 这确保每个图层在 3D 空间中正确分离，便于处理复杂的多图层设计。
✓ 增强的材质显示
自动增强材质以在 Blender 视口中更好地显示颜色和渐变。 插件智能处理导入的材质，确保最佳的可见性和渲染质量。
✓ 灵活的单位系统
支持多种单位系统以实现精确缩放：
• 点（默认）- AI/PDF 标准（72 点 = 1 英寸）
• 毫米 (mm)
• 厘米 (cm)
• 米 (m)
• 英寸
• 自定义缩放系数以实现精确控制
✓ 简化的工作流程
拖放支持允许您通过简单地将文件拖入 Blender 的 3D 视口来快速导入文件。 一次批量导入多个文件，实现高效工作流程。
✓ 高级图像处理
自动从 .ai 文件中提取嵌入图像，并将 Base64 编码的图像转换为单独的图像文件，确保在 Blender 中正确处理纹理。

概述
概述
此插件使 Blender 能够导入 CorelDRAW (.cdr) 文件，同时保留颜色、渐变、图像和图层结构。它使用内置的 Inkscape 将 CDR 文件转换为 SVG 格式，然后使用内置的 SVG 导入器将其导入 Blender。
安装
安装

1. 下载插件 zip 文件
2. 打开 Blender
3. 前往 编辑 > 偏好设置 > 插件
4. 点击"安装..."并选择 zip 文件
5. 通过勾选旁边的复选框启用插件： "导入 CorelDRAW (.cdr) 带颜色"
6. 确保内置的 "导入-导出: Scalable Vector Graphics (SVG)" 插件已启用
系统要求
系统要求
• Blender 3.0 或更高版本
• 内置 SVG 导入器 io_curve_svg 必须启用
• 插件包含内置的 Inkscape 转换器（无需额外安装）
使用方法
使用方法
导入文件
方法一：文件菜单
1. 前往 文件 > 导入 > CorelDRAW 图形 (.cdr)
2. 选择一个或多个 CDR 文件
3. 在对话框中配置导入选项
4. 点击"导入 CDR 文件"
方法二：拖放导入
1. 将 CDR 文件拖入 Blender 的 3D 视口
2. 在弹出的对话框中配置导入选项
3. 点击"确定"导入
导入选项
增强材质显示: 自动增强材质以在视口中更好地显示颜色和渐变
图层偏移: 根据图层顺序添加小的 Z 轴偏移以防止重叠
偏移距离: 设置图层之间的 Z 轴偏移距离（默认：0.001）
导入单位: 选择导入文件的单位系统：
• 毫米 (mm) - 默认，CDR 文件中常用
• 厘米 (cm)
• 米 (m)
• 英寸
• 点（72 点 = 1 英寸）
自定义缩放: 自定义缩放系数（选择自定义单位时）
偏好设置
偏好设置
通过以下路径访问： 编辑 > 偏好设置 > 插件 > 导入 CorelDRAW (.cdr) 带颜色
默认导入选项：
• 增强材质显示: 启用/禁用自动材质增强
• 导入单位: 设置默认导入单位系统
功能
功能
颜色保留
CDR 文件中的颜色会被保留并自动增强，以在 Blender 中实现最佳显示。插件创建发光着色器以确保颜色在视口和渲染中均可见。
渐变支持
导入过程中渐变会被保留并在 Blender 中正确显示。插件增强渐变材质以获得更好的可见性。
图像导入
CDR 文件中的嵌入图像会自动从 base64 编码中提取并作为单独的图像资源导入。转换过程中图像存储在临时目录中，并在导入的 SVG 中正确引用。
图层偏移
为防止图层重叠，插件可以根据图层顺序自动应用小的 Z 轴偏移。这确保每个图层在 3D 空间中略微分离。
批量导入
一次导入多个 CDR 文件。所有文件将使用相同的导入设置，您将收到导入对象和图像的摘要。
单位转换
插件支持多种单位系统，并自动将 CDR 单位转换为 Blender 单位（米）。这确保导入图形的精确缩放。
故障排除
故障排除
问题: "Parser not found" 错误
解决方案: 确保插件已正确安装。内置的 Inkscape 转换器应包含在 Inkscape 文件夹中。
问题: "Please enable SVG importer" 错误
解决方案: 前往 编辑 > 偏好设置 > 插件 并启用"导入-导出: Scalable Vector Graphics (SVG)"。
问题: 颜色显示不正确
解决方案: 在导入选项或偏好设置中启用"增强材质显示"。这确保颜色在视口中可见。
问题: 图层重叠
解决方案: 在导入选项中启用"图层偏移"以自动在 Z 轴上分离图层。
问题: 导入的对象太大/太小
解决方案: 调整"导入单位"设置以匹配 CDR 文件中使用的单位系统。CDR 文件通常使用毫米。
技术细节
技术细节
• 插件使用 Inkscape 的命令行界面将 CDR 转换为 SVG
• SVG 文件在导入过程中临时创建，之后自动清理
• Base64 编码的图像会被提取并保存为单独的文件
• 材质节点会自动创建并增强以获得更好的显示效果
• 插件支持 Blender 3.0+ 并使用现代 Python 特性
限制
限制
• 复杂的 CDR 功能可能不完全支持
• 某些 CDR 特有的效果在转换过程中可能会被简化
• 非常大的文件可能需要更长的处理时间
