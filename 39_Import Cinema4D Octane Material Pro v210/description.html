简单步骤：工作流指南

标准 FBX 工作流

  1. 在 C4D 中完成 Octane 材质设置。

  2. 从 C4D 导出 FBX 文件。

  3. 将 FBX 导入到 Blender。

  4. 选择导入的对象，打开 Import Octane Material Pro 面板，点击"导入 Octane 材质"。

  5. 验证结果，如有必要进行微调。

使用 SyncTools 的加速工作流

  1. 确保 SyncTools 在 C4D 和 Blender 中都已运行。

  2. 在 C4D 中复制选中的资产。

  3. 直接粘贴到 Blender 中。

  4. 材质将被即时导入并转换。

重要注意事项

  • 不保证完美的 1:1 匹配：由于引擎差异，可能会出现细微变化。

  • 复杂材质可能需要微调：特别是那些大量使用程序化设置的材质。

  • 建议：烘焙复杂的程序化纹理以获得最佳视觉准确度。

  • 支持的 Blender 版本：2.93 → 4.3+

  • 包含：配套 C4D 导出脚本

  • 未来更新：包含在内并持续进行！
