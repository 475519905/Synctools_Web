Import Cinema4D Octane Material Pro v2.10 - 文档

使用步骤

方法 1：使用 FBX 导出

  1. 从 C4D 导出场景：

    • 在 Cinema 4D 中打开您的项目，确保所有 Octane 材质已正确设置。
    • 将场景导出为 FBX 文件：
      • 前往 文件 > 导出，选择 FBX 文件。
      • 确保导出设置包含几何体、材质和纹理路径。

  2. 导入到 Blender：

    • 打开 Blender 并加载您的项目或创建新场景。
    • 导入 FBX 文件：
      • 前往 文件 > 导入 > FBX，然后选择并加载您导出的文件。

  3. 运行插件：

    • 确保 Import Octane Material Pro 插件已安装并启用：
      • 前往 编辑 > 偏好设置 > 插件，然后安装并启用该插件。
    • 在 Blender 中打开插件面板，点击 "导入 Octane 材质"。插件将自动匹配几何体和 Octane 材质。

  4. 检查与调整：

    • 验证材质和纹理是否正确应用。
    • 根据需要进行进一步调整，例如调整灯光或微调着色器。

方法 2：使用 SyncTools 快速导入（推荐）

  1. 启用 SyncTools：

    • 确保 SyncTools 插件已在 C4D 和 Blender 中安装并运行。

  2. 在 C4D 中复制：

    • 选择要传输的 3D 资产及其 Octane 材质。
    • 右键点击并选择 "复制"，或使用相应的快捷键复制选中内容。

  3. 在 Blender 中粘贴：

    • 在 Blender 中打开 SyncTools 面板。
    • 点击 "粘贴" 或使用相应的快捷键。3D 资产将被导入到 Blender 中，包含完整的几何体、材质和纹理。

  4. 快速检查：

    • 检查导入的资产，确保材质和几何体完好无损。
    • 如有小问题，只需手动进行微调即可。
