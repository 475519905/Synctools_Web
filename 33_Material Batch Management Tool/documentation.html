Material Manager 安装指南

系统要求

  • Blender 4.2 或更高版本
  • 操作系统：Windows、macOS 或 Linux
  • 推荐最低 8GB 内存
  • 显卡需支持 OpenGL 4.3

    • 安装方法

    方法一：安装 via Blender 偏好设置 (Recommended)

  • 下载 the material_manager.zip file
  • Keep the zip file as-is, do not extract it
  • 制作 sure the file extension is .zip
  • 打开 Blender 并进入首选项
  • Click 编辑 in the top menu
  • Select 偏好设置
  • Or use keyboard shortcut Ctrl + Alt + U (Windows/Linux) or Cmd + , (macOS)
  • 安装 the 插件
  • In 偏好设置, select the 插件 tab
  • Click 安装 button at the top
  • Navigate to and select the downloaded material_manager.zip file
  • Click 安装 Add-on
  • 启用插件
  • 搜索 for "Material Manager" in the add-ons search box
  • Check the checkbox next to the add-on to enable it
  • Wait for the add-on to initialize

    • 方法二：手动安装

    1. 下载 the material_manager.zip file
  • 找到您的 Blender 插件目录：
  • Windows: C:\Users\{USERNAME}\App数据\Roaming\Blender Foundation\Blender\{VERSION}\scripts\addons
  • macOS: /Users/{USERNAME}/库/应用 支持/Blender/{VERSION}/scripts/addons
  • Linux: ~/.config/blender/{VERSION}/scripts/addons
  • 解压 zip 文件
  • Extract the contents into the addons directory
  • Ensure the files are in a folder named material_manager
  • 重启 Blender 并启用插件
  • Follow steps 3-4 from 方法 1

    • 验证

    After installation:
  • Press N in the 3D Viewport to open the sidebar
  • Look for the "Material Manager" tab
  • If visible, the installation was successful

    • Troubleshooting

    Common Issues and Solutions

    插件 Not Appearing

  • 验证 Blender version compatibility
  • Ensure the zip file wasn't extracted before installation
  • Try restarting Blender

    • 安装 Failed

  • 验证 you have write permissions in the add-ons directory
  • Close Blender and try installing again
  • Check if antivirus is blocking installation
