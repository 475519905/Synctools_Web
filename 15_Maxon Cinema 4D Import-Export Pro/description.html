Maxon Cinema 4D 导入-导出 Pro

🚀 Maxon Cinema 4D 导入-导出 Pro
将Cinema 4D项目导入Blender的最快方式！
一款精致的、可用于生产的插件，可以直接从Blender导入选定的 .c4d .c4d项目文件——无需保持Cinema 4D打开。它以最小的摩擦保留核心资产：
Pro功能：支持导入高级Cinema 4D功能
• XPresso – Cinema 4D的节点式编程系统
• 克隆器 – MoGraph克隆器对象和实例
• 效果器 – MoGraph效果器系统（随机、延迟、公式等）
• 矩阵动画 – MoGraph矩阵模式和动画
• 动力学 – 刚体、软体和布料模拟
• 字段 – 字段系统和力场控制
概述
概述
将Cinema 4D场景直接导入Blender——并将其导出回去——无需使用额外软件或手动转换。导入导出 Pro出现在Blender的文件菜单中，让您可以打开、编辑和重新保存原生 .c4d 项目，同时保留几何体、材质、动画、摄像机、灯光、样条线、层级和缩放。非常适合需要快速周转的工作室、在工具之间切换的自由职业者，以及需要最简单流程的教育者。
功能
主要优势
双向传输 在Blender中打开.c4d文件，并从同一面板将Blender场景导出回.c4d。
完整保真传输 网格、材质与纹理、UV、骨骼、形态键、灯光、摄像机、样条线、关键帧——一切完整保留。
两种转换模式 精确模式 用于像素精确的资产， 快速模式用于快速迭代。随时切换。
智能缩放 自动单位匹配，保持两个DCC之间的尺寸一致。
自定义缓存和日志 选择临时文件的存储位置，并保留详细日志以供流程审计。
跨平台 支持Windows、macOS和Linux，兼容Blender 3.6 LTS – 5.0+.
界面快捷方式 可选的"导入.c4d"按钮位于3D视口标题栏中，支持拖放操作。
本地化支持 已包含英文和简体中文；通过一个JSON文件即可添加更多语言。

安装
安装
1. 从右侧栏下载 .zip 文件。
2. Blender → 编辑 ▸ 偏好设置 ▸ 插件 ▸ 安装 → 选择zip文件 → 启用。
3. 您将在 文件 ▸ 导入/导出 下看到 Maxon Cinema 4D (.c4d) ——或直接将.c4d文件拖入视口。
4. 导出时，选择 文件 ▸ 导出 ▸ Maxon Cinema 4D (.c4d)。
