🚀 Maxon Cinema 4D 导入‑导出 Pro 文档
将Cinema 4D项目导入Blender的最快方式！
一款精致的、可用于生产的插件，可以直接从Blender导入选定的 .c4d .c4d项目文件——无需保持Cinema 4D打开。它以最小的摩擦保留核心资产：

概述
1. 这是什么？
Maxon Cinema 4D 导入‑导出 Pro 让您可以直接在Blender内打开、编辑和保存原生.c4d场景。 几何体、材质、动画、灯光、摄像机和样条线都可以通过一次点击双向传输，因此您可以在流程中保留Cinema 4D资产，而无需启动Cinema 4D。
非常适合迁移到Blender的工作室、接收C4D文件的自由职业者，以及处理多‑DCC项目的任何人。
功能
2. 核心功能
• 双向支持 – 从文件 ▶ 导入导出菜单导入和导出.c4d文件。
• 双转换模式
- Precise （Alembic‑基）用于最高保真度。
- 标准 (FBX‑基）用于极速转换。
• 组件开关 – 精确选择要包含的元素：模型、灯光、摄像机、样条线、动画、材质。
• Drag‑and‑Drop – 将任何.c4d文件拖放到3D视口中即可获得选项弹窗。
• 一键工具栏按钮‑ – 可选的导入图标位于3D视口标题栏中，访问更快捷。
• 帧范围导入‑ – 仅导入您需要的动画片段（精确模式）。
• 自定义缓存和日志 – 设置您自己的临时目录，并随时查看详细日志。
• 跨平台‑ – 支持Windows 10/11、macOS 12+和现代Linux发行版。
• Blender 4.2 LTS就绪 – 已在最新的Blender版本上完全测试。
设置
3. 安装
1. 从BlenderMarket右侧栏下载ZIP文件（不要解压）。
2. 在Blender中：编辑 ▶ 偏好设置 ▶ 插件 ▶ 安装，然后选择ZIP文件。‑
3. 勾选启用 "Maxon Cinema 4D 导入‑导出 Pro."
4. （可选）点击插件的 ⚙‑ 偏好设置进行配置：
- 设置自定义缓存目录（大型场景推荐使用SSD）。
- 显示或隐藏3D视口导入按钮。
- 选择默认的导入/导出格式。
5. 保存偏好设置，如有提示请重启Blender。
快速入门
4. 快速入门
导入.c4d场景：
1. 文件 ▶ 导入 ▶ Maxon Cinema 4D (.c4d)
2. Choose Precise （Alembic) or 标准（FBX).
3. 切换您要导入的场景元素。
4. （精确模式）如果只需要部分动画，可指定自定义帧范围。
5. 点击导入 — 完成！
导出回.c4d：
1. 文件 ▶ 导出 ▶ Maxon Cinema 4D (.c4d)
2. 选择要导出的物体。
3. 选择精确或标准模式及要包含的组件。
4. 点击导出并选择目标位置。
提示： 您可以混合使用不同模式。例如，使用快速标准导入进行外观开发，然后切换到精确模式进行最终动画交接。‑and‑match modes. For example, use fast 标准 imports for look‑‑

兼容性
6. 兼容性
• Blender: 4.2 LTS或更新版本（也可在Alpha和Beta版本上运行）。
• 文件类型： R21及以上版本的原生.c4d文件。
• OS: Windows, macOS (Apple Silicon & Intel), Linux (glibc 2.35+).
• 取消勾选除摄像机外的所有选项。
限制
7. Known 限制
• Cinema 4D的程序化或第三方着色器可能会以烘焙纹理形式导入。‑
• MoGraph效果器和X粒子目前会被烘焙为几何体。‑
• 超大场景（1000万+多边形）可能需要精确模式和SSD缓存以获得最佳稳定性。
