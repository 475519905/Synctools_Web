MIDI Piano Motion Tools Pro

🎹 MIDI Piano Motion Tools Pro

版本: 2.0 | Blender 4.2+ | 类别: 动画

🧠 功能介绍

MIDI Piano Motion Tools Pro 可将您的 MIDI 文件转换为 Blender 内完整动画的 3D 钢琴演奏。 它支持力度感应琴键运动、动态灯光、踏板和琴锤动画，甚至基于物理的落球效果。 无论您是为 YouTube 创作内容、在线教授钢琴还是制作音乐可视化，这款插件都能处理复杂性—让您专注于创意。

PRO MAX 版本:https://superhivemarket.com/products/midi-piano-motion-tools-pro-max---transform-music-into-visual-magic




🔧 主要功能

✅ 自动 MIDI 动画系统

  • 由 MIDI 力度驱动的帧精确琴键动画

  • 逼真的琴锤和踏板运动（制音踏板、持续音踏板、柔音踏板）

  • 可配置的动画时间、力度灵敏度和帧偏移

  • 完全支持弯音、调制轮、移调和通道过滤

💡 动态视觉效果

  • 响应式琴键发光

  • 分手颜色控制和可自定义的发光强度

  • 每个 MIDI 音符的自定义琴键颜色映射

  • 基于渐变的颜色系统，可控制方向和插值


🔴 基于物理的落球效果

  • 模拟球体以精确的重力和弹跳落在琴键上

  • 发光动画撞击效果、淡出和可自定义的轨迹

  • 完全可调：大小、弹跳高度、时间偏移和发光




🎼 高级 MIDI 控制

  • 选择特定音轨和通道

  • 仅从 MIDI 文件导入选定的时间范围

  • 实时 MIDI 信息面板显示 BPM、音符数量、时长和每通道统计

🛠️ 实时编辑工具

  • 3D 视图侧边栏面板 (View3D > Sidebar > Piano)

  • 对象映射实时测试（琴键、琴锤、踏板）

  • 清除/reset all animations and materials 只需一键

  • UI 列表和取色器工具，用于管理自定义琴键颜色

📦 安装

  1. 下载 ZIP 文件

  2. 安装 via Blender 偏好设置 > 插件 > 安装

  3. 启用插件

  4. 在 3D 视图侧边栏中打开 Piano 标签

🖥️ 系统要求

  • Blender 3.6 or later

  • 兼容的 3D 钢琴模型（不包含在内）

  • 标准 MIDI file (.mid)

  • Mido 库（已捆绑在 libs 文件夹中）
