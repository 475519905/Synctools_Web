SyncTools Pro For Rhino

SyncTools Pro for Rhino - 无缝资产传输

概述

SyncTools for Rhino 是一款专业级工具，旨在简化不同 Rhino 文档之间复制和粘贴 3D 对象的流程。只需一键，即可将选中的对象从一个场景传输到另一个场景，同时保持所有几何属性和层级关系。此工具可显著提升您的 Rhino 工作流程效率。

Rhino 安装与使用

1. 插件安装

  1. 下载插件文件 SyncTools_Rhino.py
  2. 将文件复制到 Rhino 的 Python 脚本目录
  3. 在 Rhino 中运行脚本
  4. 工具会自动注册并创建浮动窗口

2. 工作流程

复制操作

  1. 在场景中选择要复制的对象
  2. 在 SyncTools 窗口中点击 "复制"
  3. 插件将您的选择导出到指定的缓存文件夹

粘贴操作

  1. 切换到目标 Rhino 文档
  2. 在 SyncTools 窗口中点击 "粘贴"
  3. 插件自动导入最新的缓存文件
