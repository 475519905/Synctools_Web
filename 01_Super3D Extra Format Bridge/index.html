Super3D Extra 格式 Bridge

导入和导出超过20种 Blender 原生不支持的3D文件格式。

此插件在 Blender 的文件 → 导入和文件 → 导出菜单中添加了额外选项，让用户能够打开和保存更多常见的3D格式文件。

支持的格式




DCC / 动画

  • Cinema 4D (.c4d)

  • 3ds Max (.max)

  • Maya ASCII (.ma)

  • Maya Binary (.mb)

  • Houdini (.hip / .hiplc / .hipnc)

  • Fusion 360 (.f3d)

CAD / 工程

  • AutoCAD DWG (.dwg)

  • STEP (.step / .stp)

  • IGES (.iges / .igs)

  • IFC (.ifc)

  • Parasolid X_T (.x_t)

  • JT (.jt)

其他3D格式

  • SketchUp (.skp)

  • 3D Studio (.3ds)

  • 3MF (.3mf)

  • APX (.apx)

  • 3DXML (.3dxml)

工作原理


导入

  1. 从文件 → 导入中选择支持的文件格式。

  2. 插件将文件上传至 Super3D 转换服务。

  3. 文件被转换为 OBJ 格式并导入当前 Blender 场景。

导出


  1. 从文件 → 导出中选择支持的格式。

  2. 场景被导出为 OBJ 格式。

  3. OBJ 文件通过 Super3D 服务转换为所选格式并保存至本地。

核心功能

  • 支持超过20种常见3D文件格式

  • 与 Blender 原生导入/导出菜单深度集成

  • 单一轻量级插件文件（.py）

  • 兼容 Blender 4.0 及更高版本

  • 可配置的 API 超时时间（30秒至2小时）

  • 可选择保留或自动删除临时文件

  • 除 Blender 内置 Python 外无需任何外部依赖

安装方法

  1. 下载插件 .zip 文件。

  2. 打开 Blender，进入编辑 → 偏好设置 → 插件。

  3. 点击安装…并选择下载的 .zip 文件。

  4. 在插件列表中启用该插件。

系统要求

  • Blender 4.0 或更高版本
