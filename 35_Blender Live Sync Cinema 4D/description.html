Blender Live Sync for Cinema 4D — 插件概述

Blender Live Sync for Cinema 4D 是一款实时同步插件，旨在简化 Cinema 4D (C4D) 与 Blender 之间的工作流程。它实现近零延迟更新，让艺术家可以在 C4D 中建模、变形、雕刻、绑定和动画，同时在 Blender 中即时看到结果——彻底消除重复的导出和导入操作。

🚀 实时同步

在 Cinema 4D 中工作，即时在 Blender 中看到更改。在 C4D 中对模型进行变形、雕刻和骨骼绑定，实时更新反映在 Blender 中，加快迭代速度，让制作流程更顺畅。


✅ 广泛的功能支持

该插件支持制作流水线中常用的多种数据类型和对象：


模型数据

实时传输核心网格信息，包括：

  • 顶点和拓扑

  • UV、法线等

变形器

支持所有影响几何变形的变形设备，包括：

  • 变形器、场/域及相关系统

生成器

C4D 生成器生成的几何体实时更新，使程序化工作流完全支持同步。

Motion图形（MoGraph）

同步Motion图形对象，如：

  • 克隆器

  • 矩阵

  • 其他 MoGraph 元素

骨骼 / 绑定

实时同步骨骼变形，实现高效的绑定和动画工作流。

雕刻 / 刻画

通过雕刻（刻画笔刷）修改的模型在 Blender 中即时更新，保持雕刻工作流快速响应。

摄像机同步

同步摄像机参数，包括：

  • 位置和旋转

  • 焦距及其他摄像机设置

时间线同步




保持两个应用程序同步：

  • 当前帧

  • 起始帧和结束帧

  • 帧率同步

⚡ 高性能




由共享内存技术驱动，插件仅传输发生变化的数据——实现极快的性能，延迟几乎为零。

🧭 可配置的轴映射

自定义从 C4D 到 Blender 的坐标系转换，支持：

  • 轴映射调整

  • 缩放

  • 偏移控制
