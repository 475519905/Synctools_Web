Blender Live Sync for Cinema 4D


安装与设置指南

Blender Live Sync for Cinema 4D 是一款连接 Cinema 4D (C4D) 与 Blender 的实时同步插件，可对网格、变形器、生成器、MoGraph 对象、骨骼、雕刻更改、摄像机和时间线设置实现近零延迟更新。

本文档说明如何在两端安装插件以及如何建立首次成功连接。

1. 系统要求

支持的软件

  • Cinema 4D：R25 / 2023 / 2024 或更新版本（推荐）

  • Blender：3.x / 4.x（推荐）

操作系统

  • Windows / macOS （取决于您的插件构建版本）

推荐设置

为获得最佳性能和最低延迟，请在同一台电脑上运行 Cinema 4D 和 Blender。
（某些版本可能支持网络同步，具体取决于您的发布版本。）

2. 软件包内容

您的下载包通常包含：

  • Cinema 4D 插件文件夹

    • 示例：BlenderLiveSync_C4D/

  • Blender 插件

    • 示例：BlenderLiveSync_Blender.zip 或 BlenderLiveSync.py

  • 可选文档文件：

    • README.txt、Manual.pdf 等

⚠️ 注意：文件夹名称可能因插件版本而有所不同。

3. 安装 Cinema 4D 插件（C4D 端）

方式 A（推荐）：安装到 C4D 的 plugins 文件夹

  1. 关闭 Cinema 4D（如果正在运行）。

  2. 找到您的 Cinema 4D 安装目录：

Windows（示例）：
C:\Program Files\Maxon Cinema 4D 2024\plugins\

macOS（示例）：
应用/Maxon Cinema 4D 2024/plugins/

  1. 将插件文件夹复制到 plugins 目录。

示例：
BlenderLiveSync_C4D/ → .../Cinema 4D/plugins/

  1. 启动 Cinema 4D。

  2. 确认插件出现在以下位置之一：

  • 扩展菜单

  • 插件菜单

  • 专用插件面板 / 工具栏

方式 B：安装到用户插件文件夹（无需管理员权限）

如果无法访问系统安装目录：

  1. 打开 Cinema 4D

  2. 前往：编辑 → 首选项

  3. 找到：插件

  4. 找到用户插件路径

  5. 将插件文件夹复制到该目录

  6. 重启 Cinema 4D

4. 安装 Blender 插件（Blender 端）

  1. 打开 Blender

  2. 前往：编辑 → 首选项 → 插件

  3. 点击安装…

  4. 选择插件文件：

    • 如果收到 .zip，直接安装 .zip（请勿解压）

    • 如果收到 .py，选择 .py 文件

  5. 安装后，搜索插件名称：
    Blender Live Sync for Cinema 4D

  6. 勾选复选框启用插件

启用后，插件界面通常显示在：

  • 3D 视口 → 侧边栏（按 N 键）

  • 或 Blender 界面中的插件选项卡（取决于版本）

5. 首次连接设置

两端插件均安装完成后，按以下步骤建立连接。

步骤一：在 Cinema 4D 中启动同步

  1. 打开您的 C4D 场景

  2. 从扩展 / 插件打开插件面板

  3. 点击以下之一（名称可能有所不同）：

  • 开始

  • 连接

  • 启用同步

  • 实时同步开启

  1. 确认插件状态显示类似以下内容：

  • 运行中

  • 等待 Blender

  • 已连接

步骤二：在 Blender 中启动同步

  1. 打开 Blender

  2. 打开插件面板（通常在 N 面板中）

  3. 点击：

  • 连接

  • 开始同步

  • 接收

  1. 如果插件支持连接设置，请确认：

  • 连接模式 = 共享内存

  • 状态 = 已连接

6. 验证同步

连接后，使用以下测试确认一切正常。

6.1 网格更新测试

  1. 在 Cinema 4D 中创建一个立方体（或任意基本体）

  2. 移动、旋转或缩放它

  3. 对象应立即在 Blender 中更新

6.2 拓扑 / UV / 法线测试

  1. 在 C4D 中将基本体转换为可编辑网格

  2. 修改点 / 边 / 多边形

  3. 确认 Blender 正确更新网格形状

6.3 变形器测试

  1. 在 C4D 中添加变形器（弯曲 / 扭曲等）

  2. 调整变形器强度和设置

  3. Blender 应实时更新

6.4 生成器 & MoGraph 测试

  1. 在 C4D 中创建生成器或 MoGraph 对象

  2. 更改其参数（克隆数量、间距等）

  3. Blender 应更新生成的几何体

6.5 骨骼变形测试

  1. 在 C4D 中将骨骼绑定到网格

  2. 移动/旋转骨骼

  3. Blender 应即时更新变形

6.6 摄像机同步测试

  1. 在 C4D 中选择一个摄像机

  2. 调整：

    • 位置 / 旋转

    • 焦距

  3. Blender 摄像机应匹配这些变化

6.7 时间线同步测试

  1. 在 C4D 中拖动时间线或点击播放

  2. 确认 Blender 同步了：

    • 当前帧

    • 起始帧 / 结束帧

    • 帧率（FPS）

7. 轴映射设置（坐标转换）

Cinema 4D 和 Blender 使用不同的坐标系。
如果您注意到方向不正确（轴翻转、旋转错误、缩放不匹配），请配置轴映射：

  1. 打开插件设置（C4D 端或 Blender 端，取决于版本）

  2. 找到轴映射 / 坐标转换

  3. 调整：

  • 轴转换（X/Y/Z 映射）

  • 缩放

  • 偏移

提示：如果您的模型出现旋转 90° 或镜像，通常是轴映射的问题。

8. 故障排除

问题：插件未出现在 Cinema 4D 中

解决方案：

  • 确认插件文件夹放置在正确的 plugins 目录中

  • 确保文件夹嵌套正确
    （示例：避免 plugins/BlenderLiveSync/BlenderLiveSync/）

  • 重启 Cinema 4D

  • 尝试通过用户插件路径安装

问题：插件未出现在 Blender 中

解决方案：

  • 确保在首选项 → 插件中已启用

  • 使用搜索栏输入 Live Sync

  • 检查 3D 视口中的 N 面板

问题：无连接 / 无更新

解决方案：

  • 确保两端插件均在运行

  • 重启 Cinema 4D 和 Blender

  • 尝试重新点击连接 / 开始同步

  • 确认插件使用共享内存模式（如果可用）

问题：延迟或性能缓慢

解决方案：

  • 减少同步对象数量

  • 在大量编辑时避免同步极高多边形数的雕刻网格

  • 使用共享内存模式以获得最佳速度

  • 仅同步变化的数据（推荐工作流）

问题：Blender 中的缩放或旋转错误

解决方案：

  • 配置轴映射

  • 应用缩放/偏移设置

  • 确认两个场景使用一致的单位
