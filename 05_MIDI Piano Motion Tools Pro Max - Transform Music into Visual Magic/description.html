✅主要功能

🎹A 完整的钢琴动画系统

  • 自动钢琴琴键动画，具有逼真的节奏

  • 与音符同步的琴锤动画

  • 踏板动画（延音、柔音、持续）

  • 支持完整的 88 键范围（MIDI 21–108）。



🎨高级视觉效果

  • 动态按键发光效果，颜色可自定义

  • 基于速度和彩虹渐变的颜色模式

  • 每个通道和每个按键的颜色均可自定义。

  • 球下落可视化

  • 完整的 16 通道 MIDI 颜色管理


🎵音频集成

  • 内置SF2 音色库 用于音频生成

  • 按 MIDI 通道导出音频

  • 支持自定义音色库

  • 高质量 FluidSynth 渲染

  • 多种插值模式（线性、常量、贝塞尔、智能）。

  • 速度感应动画时长

  • 视觉序列编辑器集成

🔧高级自定义

  • 自定义琴键命名规则

✅工作流程

  1. 导入内置钢琴模型（或使用您自己的钢琴模型）。

  2. 通过钢琴面板加载您的 MIDI 文件

  3. 配置按键、琴锤、动画、特效和颜色。

  4. 一键生成动画

  5. 根据需要调整通道和设置

  6. 使用 SF2 音色库导出音频

  7. 渲染您最终的钢琴演奏动画
