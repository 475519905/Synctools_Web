Maxon Cinema 4D 文件 导入-导出 (.c4d 文件) - 文档

📦 插件 文档
Maxon Cinema 4D 文件导入-导出（.C4D 文件）插件由 blenderVfx Master 开发，允许用户在 Blender 中直接导入和导出 .c4d 项目文件，实现 Blender 与 Cinema 4D 之间的无缝集成。该插件无需安装 Cinema 4D，简化了跨平台工作的艺术家和工作室的工作流程。
安装
🔧 安装指南
下载插件：
从 Superhive 产品页面获取 .zip 文件。
在 Blender 中安装：
1. 打开 Blender。
2. 导航到 编辑 > 偏好设置 > 插件。
3. 点击 安装，选择已下载的 .zip 文件并确认。
启用插件：
1. 在插件列表中搜索 Maxon Cinema 4D 文件导入-导出。
2. 勾选插件旁边的复选框以激活。
导入
📥 导入 .c4d 文件
1. 在 Blender 中，前往 文件 > 导入 > Maxon Cinema 4D 文件 (.c4d)。
2. 浏览并选择要导入的 .c4d 文件。
3. 在导入选项中，选择要包含的资产类型（例如模型、动画、材质）。
4. 点击 导入 Maxon Cinema 4D 文件 开始导入过程。
注意： 导入大型 .c4d 文件可能需要一些时间，请耐心等待。
导出
📤 导出 .c4d 文件
1. 在 Blender 中，导航到 文件 > 导出 > Maxon Cinema 4D 文件 (.c4d)。
2. 选择导出 .c4d 文件的目标路径和文件名。
3. 在导出选项中，选择要包含在导出中的资产类型。
4. 点击 导出 Maxon Cinema 4D 文件 开始导出过程。
注意： 插件会将 Blender 资产转换为 .c4d 格式，使其与 Cinema 4D 兼容。
重要事项
⚠️ 重要注意事项
• 请确保使用正版 Blender 以获得最佳兼容性。
• 虽然插件支持多种类型的资产，但某些复杂的材质或动画在导入/导出后可能需要调整。
• 该插件无需安装 Cinema 4D 即可运行，简化了 Blender 和 Cinema 4D 文件之间的工作流程。
