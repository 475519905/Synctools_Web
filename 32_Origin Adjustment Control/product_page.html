描述

Origin Adjustment Control


概述

此插件增强了在 Blender 中精确操控对象原点的能力。它支持对齐原点、将原点吸附到曲面、应用变换、管理预设等操作。界面包含模态交互、面板控件，以及 Gizmo 显示和坐标转换等工具增强功能。


功能说明

1. 全局函数

  • get_valid_objects(context)
    获取所有类型为 MESH、CURVE、SURFACE 或 FONT 的选中对象，用于原点调整。

  • apply_origin_transform(obj, new_origin)
    在保持几何形状不变的情况下，将对象的原点变换到世界空间中的指定位置。

2. 模态操作符

  • ADJUST_ORIGIN_OT_modal：通过鼠标移动以交互方式调整对象原点。
    使用步骤：
    1. 启用自定义原点功能。
    2. 通过"调整原点模态"按钮调用操作。
    3. 使用鼠标重新定位原点；左键确认，右键取消。

3. 动作与操作符

  • APPLY_ORIGIN_OT_apply：将原点移至对象的几何中心。
    用法：在插件面板中点击"原点到中心"。

  • ORIGIN_OT_ModeSwitch：在原点控制模式之间切换：世界、局部或自定义参考。

  • OBJECT_OT_precise_rotation：根据每个轴的输入角度，对对象应用精确的旋转值。

4. 自定义原点面板（界面）

  • 位于 VIEW_3D > 侧边栏 > 原点调整。
    面板功能：
    • 通过切换开关启用自定义原点定位。
    • 手动输入精确的原点位置。
    • 调用关键原点相关操作符的按钮。

5. 主要特性

5.1. 精确变换
  • 使用属性输入（custom_origin 或 precise_rotation）直接更新原点的位置或旋转。
5.2. 原点 Gizmo
  • 通过可见的 3D 控件，提供直观且交互式的原点操控方式。
5.3. 关键帧支持
  • 使用 ORIGIN_OT_AddKeyframe 操作符添加关键帧，以记录和动画化原点变换。
5.4. 吸附系统
  • 使用 ORIGIN_OT_SnapToSurface 将原点吸附到最近的曲面点。

6. 坐标转换

  • ORIGIN_OT_CoordinateConverter 操作符支持在世界、局部和自定义空间之间转换坐标，实现灵活的原点控制。

7. 预设管理

  • 原点预设系统：
    • 使用自定义名称保存当前原点位置和旋转（ORIGIN_OT_PresetManager）。
    • 加载已保存的预设以应用一致的变换。
    • 通过保存、加载或删除操作管理预设。

8. 批量操作

对选中对象执行批量变换：

  • 对齐：将所有选中对象的原点对齐到活动对象。
  • 分布：均匀分布对象间的原点（开发中）。
  • 重置：将原点重置为世界坐标中的 (0, 0, 0)。

安装说明

安装步骤：

  1. 将 Python 脚本（*.py）保存到您的计算机。
  2. 打开 Blender，前往 编辑 > 首选项 > 插件 > 安装。
  3. 选择脚本文件并勾选复选框以启用插件。

注册与注销：

  • 注册：启用插件后自动初始化。
  • 注销：在首选项中使用禁用选项，或手动调用 unregister() 函数。

自定义属性

以下属性已添加到 Scene 对象：

  • enable_custom_origin：切换自定义原点模式。
  • custom_origin：将当前原点位置存储为 3D 向量。
  • last_origin_location：跟踪上次使用的原点位置以便快速还原。

其他属性链接到对象实例，例如用于动画的 origin_keyframes 和用于预设管理的 origin_presets。

开发说明

  • 此工具采用模块化设计，便于未来更新扩展。
  • 使用 REGISTER/UNREGISTER 块的模板结构进行安全的属性管理。
