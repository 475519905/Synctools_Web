导入 CorelDRAW 图形文件（.cdr）

导入 CorelDRAW 图形文件（.cdr）
通过与 Blender 的无缝集成，彻底改变您的 CorelDRAW 工作流！
这款强大的插件允许您将 .cdr 文件直接导入 Blender，同时完整保留颜色、渐变、图像和图层结构。
核心功能
核心功能

✓ 完整 CDR 支持
高保真导入 CorelDRAW 文件，完整保留原始设计中的所有视觉元素。
✓ 颜色与渐变保留
颜色和渐变自动保留并优化，以便在 Blender 视口中获得最佳显示效果。
✓ 图像导入
CDR 文件中的嵌入图像会被自动提取并作为独立图像资产导入。
✓ 图层偏移管理
智能图层偏移系统根据图层顺序自动应用 Z 轴偏移，防止图层重叠。
✓ 灵活的单位系统

支持多种单位系统，包括：
• 毫米
• 厘米
• 米
• 英寸
• 磅（点）
非常适合精确作业。

✓ 拖放导入
直接将 CDR 文件拖放到 Blender 的 3D 视口中即可即时导入。
✓ 材质增强
自动材质增强确保导入的图形在视口和渲染中均呈现精美效果。
✓ 批量导入
使用统一设置一次性导入多个 CDR 文件。
技术细节
技术细节
该插件使用捆绑的 Inkscape 强大转换引擎将 CDR 文件转换为 SVG 格式，然后通过 Blender 内置的 SVG 导入器导入。
这确保了最大兼容性和高质量的还原效果。
非常适合使用 CorelDRAW 的设计师、艺术家和专业人士，他们需要将矢量图形带入 Blender 用于：
• 3D 项目
• 动画制作
• 进一步编辑和精修
系统要求
系统要求
• Blender 4.2 或更高版本
• 内置 SVG 导入器 io_curve_svg 必须已启用
