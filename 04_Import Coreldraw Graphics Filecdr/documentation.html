导入 CorelDRAW 图形文件（.cdr）- 使用文档

导入 CorelDRAW 图形（.cdr）使用文档
完整指南：将 CorelDRAW 图形文件导入 Blender，完整保留颜色、渐变色及图层结构。
概述
概述
本插件可让 Blender 导入 CorelDRAW（.cdr）文件，同时保留颜色、渐变色、图像及图层结构。它使用内置的 Inkscape 将 CDR 文件转换为 SVG 格式，再通过 Blender 内置 SVG 导入器完成导入。
安装
安装
1. 下载插件 zip 文件
2. 打开 Blender
3. 前往 编辑 > 偏好设置 > 插件
4. 点击"安装…"并选择 zip 文件
5. 勾选"导入 CorelDRAW (.cdr) with 颜色"旁的复选框以启用插件
6. 确保已启用内置插件"导入-导出: 可缩放矢量图形 (SVG)"
系统要求
系统要求
• Blender 4.2 或更高版本
• 必须启用内置 SVG 导入器 io_curve_svg
• 插件内含 Inkscape 转换器（无需额外安装）
使用方法
使用方法
导入文件
方法一：文件菜单
1. 前往 文件 > 导入 > CorelDRAW 图形（.cdr）
2. 选择一个或多个 CDR 文件
3. 在对话框中配置导入选项
4. 点击"导入 CDR 文件"
方法二：拖放
1. 将 CDR 文件拖入 Blender 的 3D 视图
2. 在弹出的对话框中配置导入选项
3. 点击"确定"完成导入
导入选项
增强材质显示：自动增强材质，以便在视图中更好地显示颜色和渐变色
图层偏移：根据图层顺序在 Z 轴方向添加微小偏移量，防止图层重叠
偏移距离：设置图层间 Z 轴偏移距离（默认值：0.001）
导入单位：选择导入文件所用的单位制：
• 毫米（mm） - 默认值，CDR 文件常用单位
• 厘米（cm）
• 米（m）
• 英寸
• 点（72 点 = 1 英寸）
自定义缩放：自定义缩放比例（选择自定义单位时有效）
偏好设置
偏好设置
访问路径：编辑 > 偏好设置 > 插件 > 导入 CorelDRAW (.cdr) with 颜色
默认导入选项：
• 增强材质显示：启用/禁用自动材质增强
• 导入单位：设置默认导入单位制
功能特性
功能特性
颜色保留
CDR 文件中的颜色会被完整保留，并自动增强以在 Blender 中达到最佳显示效果。插件会创建自发光着色器，确保颜色在视图和渲染中均可见。
渐变色支持
导入时渐变色会被完整保留，并在 Blender 中正确显示。插件会对渐变材质进行增强处理以提升可见度。
图像导入
CDR 文件中的嵌入图像会自动从 base64 编码中提取，并作为独立图像资产导入。转换过程中图像暂存于临时目录，并在导入的 SVG 中正确引用。
图层偏移
为防止图层重叠，插件可根据图层顺序自动在 Z 轴方向施加微小偏移，确保每个图层在三维空间中略有分离。
批量导入
支持同时导入多个 CDR 文件。所有文件使用相同的导入设置，完成后将显示导入对象和图像的汇总信息。
单位换算
插件支持多种单位制，并自动将 CDR 单位换算为 Blender 单位（米），确保导入图形的比例准确无误。
常见问题
常见问题
问题：提示"解析器未找到"错误
解决方案：确保插件已正确安装，内置 Inkscape 转换器应位于 Inkscape 文件夹中。
问题：提示"请启用 SVG 导入器"错误
解决方案：前往 编辑 > 偏好设置 > 插件，启用"导入-导出: 可缩放矢量图形 (SVG)"。
问题：颜色显示不正确
解决方案：在导入选项或偏好设置中启用"增强材质显示"，确保颜色在视图中可见。
问题：图层重叠
解决方案：在导入选项中启用"图层偏移"，自动在 Z 轴方向分离各图层。
问题：导入对象过大或过小
解决方案：调整"导入单位"设置，使其与 CDR 文件中使用的单位制一致。CDR 文件通常使用毫米。
技术细节
技术细节
• 插件使用 Inkscape 命令行界面将 CDR 转换为 SVG
• 导入过程中会临时生成 SVG 文件，完成后自动清理
• Base64 编码的图像会被提取并保存为独立文件
• 自动创建并增强材质节点以改善显示效果
• 插件支持 Blender 3.0 及以上版本，使用现代 Python 特性
已知限制
已知限制
• 复杂的 CDR 特性可能无法完全支持
• 某些 CDR 专属效果在转换过程中可能被简化
• 超大文件处理时间可能较长
